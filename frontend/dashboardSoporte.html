Soporte: preguntas frecuentes

Este sistema funciona mediante sensores instalados en distintos puntos estratégicos dentro de un área. Cuando un sensor detecte movimiento en una zona, enviará una señal al centro de control que notificará al personal que monitorea el sistema.

El sistema permite monitorear hasta 50 sensores de manera simultánea, para tener la máxima cobertura posible dentro del área protegida.

Sí, este sistema está diseñado para ser escalable. Es posible agregar más sensores o cámaras en el futuro según las necesidades del usuario.

El sistema cuenta con una aplicación móvil que permite a los usuarios armar o desarmar la alarma, recibir notificaciones y monitorear el estado de los sensores desde cualquier lugar.
